## Projeto Aplicação de Musicas com POO - Java

Esse projeto tem o intuito de consolidar e praticar vários aspectos da orientação a objetos, como herança, polimorfismo e encapsulamento. Destacando os principais conceitos que preciso me familiarizar para trabalhar com Java ou outras linguagens orientadas a objetos.

## Resumo desse projeto

A estrutura do sistema é baseada em uma superclasse chamada Audio, que representa qualquer tipo de conteúdo sonoro. A partir dela, são criadas classes derivadas, como Musica e Podcast, aplicando os princípios de herança e encapsulamento.
